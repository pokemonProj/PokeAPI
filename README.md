### Format

```json
["slot" : 1 ,
 "type" : { 
           "name" : "normal",
           "url" : "https://pokeapi.co/api/v2/type/12/"
         }]
```

### Description

(Integer) **```slot:```** Defines 

dict **```type```**: 

* str **```name```** : Pokemon Type (String)

* link **```url```**: Pokemon Type Descpription
